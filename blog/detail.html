How to detect an cycle in graph ?

When we start from the source node, we mark it as visiting. Then go for checking its neighbors which are not visited. If anyone is visiting=> detected cycle. Then do the recursion step on them. All done mark current node as visited and return true.
here we use number 0 represent the initial state; number 1 represent visiting state; number 2 represent already visited state.

Core Code：

          
          Public boolean cycle(Map<> graph, int source, int end,int[] colors){   
              Colors[source] = 1;
              for(int next: graph.get(source)){
                if(colors[next] == 1) return false;
                if(colors[next]==0 && !cycle(graph,next,end,colors))return false;
              }
              Colors[source] = 2;
              Return true; 
          }
