Degree of an Array

testing


Input: [1, 2, 2, 3, 1]
Output: 2
Explanation: The input array has a degree of 2 because both elements 1 and 2 appear twice. Of the subarrays that have the same degree: [1, 2, 2, 3, 1], [1, 2, 2, 3], [2, 2, 3, 1], [1, 2, 2], [2, 2, 3], [2, 2] The shortest length is 2. So return 2.


The problem I meet


1. how to find the degree of this array? what if there are more than one element contribute to the max degree of this array?


2.how to find the first occurence and the last occurence of one element inside an unsorted array? (It’s pretty clear that we could use binaray search if the array is sorted)


Solutions:


For Question 1:


we could use a HashMap to count the frequence of each element. Then using the belowe function to get the degree int degree = Collections.max(count.values()); we don’t matter how many elements qualified, but we only care the degree which is the maximum number of all values


For Question 2:


we use two HashMap left and right to record the first occur and the last occur. since we record this based on the current value which is the key indeed. so we use hashMap which is perfect for put and get based on element.

CODE HERE

   
    
      Map left = new HashMap<>();
      Map right = new HashMap<>();
     Map count = new HashMap<>();
     for(int i = 0; i < nums.length; i++){
              int x = nums[i];
            if(left.get(x) == null) left.put(x,i);
            right.put(x,i);
            count.put(x,count.getOrDefault(x,0) +1);
        }
         int res = nums.length;
         int degree = Collections.max(count.values());
         for(int x : count.keySet()){
             if(count.get(x) == degree){
                 res = Math.min(res,right.get(x) - left.get(x) +1);
             }
         }        
        return res;        
    }

Another Solution

       

if (nums.length == 0 || nums == null) return 0;
        Map map = new HashMap<>();
        for (int i = 0; i < nums.length; i++){
           if (!map.containsKey(nums[i])){
               map.put(nums[i], new int[]{1, i, i});  // the first element in array is degree, second is first index of this key, third is last index of this key
           } else {
               int[] temp = map.get(nums[i]);
               temp[0]++;
               temp[2] = i;
           }
        }
        int degree = Integer.MIN_VALUE, res = Integer.MAX_VALUE;
        for (int[] value : map.values()){
            if (value[0] > degree){
                degree = value[0];
                res = value[2] - value[1] + 1;
            } else if (value[0] == degree){
                res = Math.min( value[2] - value[1] + 1, res);
            } 
        }
        return res;
    }
